Welcome To PokeTracker!

PokeTracker is a tool that I developed in order to help aspiring trainers reach their goal of completing the illusive Living Dex. I of course don't own any rights to Pokemon or anything Pokemon related, and this site is to help trainers catch 'em all.

Application Info

Technical Info

PokeTracker uses JavaScript and localStorage to populate and save data. All data is stored on your local browser, not in a database. There is no way for me or anyone else to access this data. Clearing your browsers history will probably erase all of that data. When navigating, please use the buttons on the page, as refreshing the page may cause issues.

How does PokeTracker work?

PokeTracker was designed to allow for users to track their progress as they play through the games. With PokeTracker, you can go to any route or location and see which Pokemon are available. Then the user can select which Pokemon they have caught.
